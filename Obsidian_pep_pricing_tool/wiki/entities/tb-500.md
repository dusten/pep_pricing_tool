# TB-500 / Thymosin Beta Family

**⚠ High nomenclature risk** — "TB-500" is used ambiguously across this market for BOTH the full 43aa TB4 and the 7aa fragment (B5), and vendors rarely give a CAS number to disambiguate. **Correction (2026-07-12)**: an earlier version of this page claimed "TB-500" almost always means the fragment — a close read of this project's actual vendor extraction data (35 Claude call-log rows across 18 vendors, see [[wiki/analyses/2026-07-12-tb500-b4-reidentification]]) found the opposite is true for this catalog: the large majority of vendors' plain "TB-500"/"TB500" listings are the full-length TB4, several confirmed directly from the vendor's own document text ("TB500(Thymosin B4 Acetate)"). Only vendors carrying a SEPARATE, distinctly-priced line explicitly marked Frag/B5/17-23 are the fragment. TB5 (CAS 948841-07-4) is a third, completely unrelated small molecule. CAS number is the only reliable differentiator, and even Claude's own extraction flagged this ambiguity on nearly every plain listing.

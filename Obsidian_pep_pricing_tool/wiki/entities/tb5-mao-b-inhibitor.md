# TB5 (MAO-B Inhibitor) — NOT a Peptide

Parent: [[wiki/entities/tb-500|TB-500 / Thymosin Beta Family]]

**⚠ This is a small molecule with zero structural or mechanistic relation to thymosin peptides. It shares only the "TB" label in some vendor/community contexts.**

## Identity

| Field | Value |
|-------|-------|
| IUPAC | (E)-1-(5-Bromothiophen-2-yl)-3-[4-(dimethylamino)phenyl]prop-2-en-1-one |
| CAS | 948841-07-4 |
| Formula | C₁₅H₁₄BrNOS |
| MW | 336.25 Da |
| Mechanism | Potent reversible MAO-B inhibitor (Ki ≈ 0.11 μM, ~13× selective over MAO-A) |
| Use | Neuroscience / neuroprotection / dopamine regulation research |

## Vendor Risk

At least one vendor (Biosynth) has mislabeled this compound as "a synthetic peptide derived from Thymosin Beta-4" — this is factually incorrect. CAS 948841-07-4 is not a peptide and has no thymosin-derived sequence.

**Flag any vendor listing TB5 / CAS 948841-07-4 with thymosin claims.**

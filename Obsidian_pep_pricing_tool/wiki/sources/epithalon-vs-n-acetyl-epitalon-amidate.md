# N-Acetyl Epitalon Amidate vs Epitalon Comparison

**Origin:** `raw/clippings/dustenPep-Resources.md`
**Source URL:** https://github.com/dusten/Pep-Resources/blob/main/N-AcetylEpitalonAmidate_vs_Epitalon.md

## Key takeaways

- Both share the same AEDG core sequence and mechanism (telomerase upregulation, pineal/melatonin, antioxidant)
- Ac-AEDG-NH₂ adds N-terminal acetylation + C-terminal amidation → resists exopeptidase breakdown
- Modified form: significantly higher metabolic stability, extended half-life, improved bioavailability
- Effective dose gap: base 5–10 mg/day vs modified 100–1,000 mcg/day (~10× efficiency advantage)
- Modified has no official CAS; some vendors reuse the base CAS (307297-39-8) — incorrect
- Modified form has limited direct research; benefits are mechanistically inferred from base
- Solubility of modified can be batch-dependent
